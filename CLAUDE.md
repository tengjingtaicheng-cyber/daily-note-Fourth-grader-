# CLAUDE.md

## プロジェクト概要

このフォルダは、理学療法士養成校に通う**4年生**が、**治療学臨床実習**のデータをまとめるための課題用フォルダです。

## 背景・目的

- 理学療法士（PT）を目指す学生が、臨床実習中に経験した症例や治療内容をデイリーノートとして記録・整理する
- 実習で学んだ内容（評価・治療・患者対応など）を振り返り、知識と技術の定着を図ることが目的

## 情報収集ルール

- 基本的な情報収集はWeb検索を使用する
- 検索結果はなるべく公式のソース（学会・協会・教科書出版社・大学・政府機関など）を優先して参照する

## 作業ルール

- Word（`.docx`）ファイルとMarkdown（`.md`）ファイルを自動で作成し、このフォルダ内に保存する
- Wordファイルを作成する際は、`/Users/fujiitaisei/Desktop/4年デイリーノート/1_デイリーノート(治療学臨床実習) (1)のコピー.docx` を複製し、その複製ファイルに内容を記入する
- ファイル名は `DailyNote_日付.docx` および `DailyNote_日付.md` とする（例：`DailyNote_20260509.docx`）
- Markdownファイル（`.md`）の保存先は `/Users/fujiitaisei/Desktop/4年デイリーノート/mdの保存場所/` とする
- 変更のたびに Git でコミットし、GitHub リポジトリ（`daily-note-Fourth-grader-`）へプッシュして記録を残す

## 文章作成スタイル

デイリーノートの文章は、「勉強は苦手だけど4年生まで進級してきた理学療法士の学生」が書いたようなイメージで作成する。
専門用語は適切に使う。一生懸命考えた結果として表現がやや素朴になるニュアンスは残しつつ、文章として不自然にならない形にまとめる。

### デイリーノートの記入項目（①〜⓾）

Wordファイル（`1_デイリーノート(治療学臨床実習).docx`）には以下の項目がある。
ユーザーが各項目の内容を伝えるので、それを整理してまとめ、**各項目名の右隣（同じ行）**に文章として記入する。

| 番号 | 項目名 |
|------|--------|
| ① | 見学内容 |
| ② | 情報収集 |
| ③ | 検査測定 |
| ④ | 動作観察 |
| ⑤ | 統合と解釈 |
| ⑥ | 問題点の抽出 |
| ⑦ | 目標設定 |
| ⑧ | プログラム立案 |
| ⑨ | プログラム実施 |
| ⓾ | 再評価 |

### 文章作成の具体的なルール

- ユーザーが話し言葉や箇条書きで内容を伝えたら、それをデイリーノートらしい文体に整える
- 難しすぎる表現は避け、学生らしい素直な言葉で書く
- 専門用語は使うが、多用しすぎない。使う場合も自然な流れに組み込む
- 各項目は簡潔にまとめる（長すぎない）

### 各項目の書き方ガイド（⑤〜⓾）

**重要：** ユーザーから受け取った情報が不足していて、各項目のルールに則って書けない場合は、無理に文章を作ったり嘘をついたりしない。その項目は空欄のままにして、ユーザーに「この項目を書くために〇〇の情報が必要です」と伝える。すべての項目を埋めることよりも、正確であることを優先する。

#### ⑤ 統合と解釈
情報収集・検査測定・動作観察のバラバラな情報を整理して、患者の問題の原因と構造を結びつける。ICFの枠組み（心身機能→活動→参加→環境因子）に沿って「この機能障害があるから、この活動制限が起きている」という因果関係を順に述べる。患者の強み（残存機能・感覚など）もプラス面として記載する。主観や飛躍した解釈は含めず、検査結果に基づいて書く。

#### ⑥ 問題点の抽出
統合と解釈をふまえて、介入すべき問題点を列挙する。ICFの各レベル（機能障害・活動制限・参加制約）に沿って整理し、優先度の高いものから「〜の低下」「〜の制限」という具体的な表現で書く。

#### ⑦ 目標設定
「達成できたかどうか判断できる」具体的な内容にする。短期目標と長期目標を連動させ、実習期間・回復見込み・患者のニーズと照らし合わせて現実的に設定する。

#### ⑧ プログラム立案
「どの問題点に対して」「どの目標のために」実施するかを明確にし、段階的な難易度設定を意識する。アプローチの選択根拠を簡潔に述べる。

#### ⑨ プログラム実施
実施内容・量・時間を具体的に記載する。実施中の患者の反応や変化（疼痛・疲労・動作変化）も記録する。

#### ⓾ 再評価
実施前後の変化を比較し、目標に対する進捗（達成・未達成・想定より早い進歩）を述べる。その結果からプログラムの継続・修正・終了を判断して記載する。
