## 指導を受けたこと

### トイレ動作の重要性
- 実習のゴール設定は「**歩行**」と「**トイレ動作**」の2点
- トイレ動作は毎日複数回・転倒リスク高・自尊心への影響が他ADLより大きい
- 患者側：「失敗が恥ずかしくて頼めない」「家族にも羞恥心がある」
- 家族側：繰り返し介助・腰痛・夜間起こされる・失敗時の洗濯・心理的抵抗
- 「せめてトイレだけは自立して退院してほしい」という要望は入院初期から非常に多い

### トイレ動作の17項目評価
BIの「一部介助」で終わらせず、どの項目に問題があるかを特定することでアプローチが具体化する。

| 動作の流れ |
|---|
| ① 移動（車椅子駆動） |
| ② トイレへの進入方向確認 |
| ③ 車椅子管理（ブレーキ・フットレスト） |
| ④ 立ち上がり |
| ⑤ 90度方向転換 → 手すり把持 |
| ⑥ ズボンを下げる |
| ⑦ 90度方向転換 → 着座 |
| ⑧ 排泄 |
| ⑨ 後始末（清拭） |
| ⑩ 立ち上がり → 90度方向転換 |
| ⑪ ズボンを上げる |
| ⑫ 90度方向転換 → 車椅子着座 |
| ⑬ ブレーキ・フットレスト解除 → 退出 |

**項目別アプローチの例**

| できない項目 | 必要なアプローチ |
|---|---|
| 移動・退出 | 車椅子駆動練習 |
| 進入・退出確認 | 注意・高次脳機能へのアプローチ |
| 立ち上がり・方向転換 | バランストレーニング |
| 清拭 | 上肢機能訓練・体幹回旋 |

### FIM的評価の活用
- 初期評価と経過の点数を比較 → 治療効果を数値で示せる
- 「中介助→軽介助」より「10点→50点」の方が患者・家族・チームに伝わりやすい
- 点数が上がれば患者のモチベーション向上・PT への信頼にもつながる
- 伸びていない項目はプログラムの再検討へ

| 点数 | 状態（FIM正式名称） |
|---|---|
| 7 | 完全自立 |
| 6 | 修正自立 |
| 5 | 監視・準備 |
| 4 | 最小介助 |
| 3 | 中等度介助 |
| 2 | 最大介助 |
| 1 | 全介助 |

合計点数：18〜126点（運動項目13 + 認知項目5）

### 動作別・主働筋の対応

| 動作 | 主な筋肉 |
|---|---|
| 方向転換・足踏み | 中殿筋・下腿三頭筋 |
| 中腰保持・ズボン下げ | 大腿四頭筋（遠心性）・腹直筋 |
| 着座 | 大腿四頭筋（遠心性でブレーキ） |
| 清拭・後始末 | 腹斜筋・体幹回旋・肩関節ROM |
| ズボン上げ | ズボン下げの拮抗筋 |

### 実技：ストレッチ・ROM訓練

**肩甲骨ストレッチ**
- 内側縁に指3本・外側縁に母指球を引っかけて肩甲骨を確実に捕まえる
- 水平ではなく**胸郭の丸みに沿って外転方向**に誘導（水平に引くと痛みの原因）
- 外転→戻すを繰り返した後、挙上・外転・下制・内転の**円運動**で仕上げ
- PTの立ち位置・姿勢の安定（近づく・体を安定させる）も同様に重要

**股関節ROM（1B抑制の活用）**
- 膝裏・ふくらはぎ下から支えてゆっくり持ち上げる
- エンドフィールまで押し込み数秒保持 → ゆっくり戻す
- 最後に**長軸方向へテンション**をかけ関節包・靭帯レベルまでストレッチ
- 伸筋パターンが強い場合：鼠径部で足を固定・骨盤の代償を抑え・大内転筋腱を圧迫して1B抑制をかけながら外転へ誘導

### 実技：筋力トレーニング

**SLR（Straight Leg Raise）**
- 対象：腸腰筋・腹直筋（下部）・外腹斜筋
- **下ろす時（遠心性収縮）をゆっくり・息を吐きながら**意識する
- 体幹回旋・屈伸の安定性向上

**逆SLR**
- かかとをベッドに押し付ける
- 対象：下肢後面全体（臀筋〜）
- 歩行**立脚後期（ターミナルスタンス〜プレスイング）**に直結
- 姿勢改善・方向転換の質向上にも有効

**ブリッジ運動**
- PTが麻痺側足底に圧をかけ、膝・臀部・肩が一直線になるよう介助
- 目的①：大臀筋強化
- 目的②：**麻痺側骨盤の後方回旋→前方回旋への誘導**（同じ動作でもPTの目的・声かけで効果が変わる）
- 麻痺側のみで行うとより直接的な効果

**キッキング**
- 股関節屈曲・膝屈曲・足背屈からスタートし、すべて伸展させる
- **足裏より膝裏への抵抗が強くなる**のが正しい形
- 手の甲をふくらはぎに当てて抵抗をかける

**ハーフスクワット**
- 臨床ではフルスクワット不可（膝への負担大）
- 膝とつま先が一致する深さが目安
- 大腿四頭筋・殿筋・脊柱起立筋を同時強化
- **麻痺側に重心を置きPTが骨盤を介助** → 筋緊張の正常化・協調性改善

## 調べたこと

- **BIとFIMの違い**：BIはADL10項目・最大100点。FIMは18項目（運動13+認知5）・7段階評価・合計18〜126点。FIMの方が細かく変化を捉えやすい
- **1B抑制（自原抑制）のメカニズム**：筋腱への**張力（伸張）刺激** → ゴルジ腱器官 → Ib求心性線維 → 脊髄介在ニューロン → 抑制性介在ニューロン → α運動ニューロン抑制 → 当該筋弛緩（拮抗筋は興奮）
- **遠心性収縮**：筋が伸張しながら収縮する形態。着座や下り動作でのブレーキとして働く。求心性収縮より筋への負荷が大きい
- **片麻痺の伸筋パターン**：下肢伸展・内転・内旋が典型的な痙縮パターン。麻痺側骨盤の後方回旋を伴うことが多い
- **中殿筋の機能**：股関節外転・骨盤の側方安定。歩行立脚期のトレンデレンブルグ徴候と関係

## 感想・反省

- トイレ動作がこれほど多くの要素に分解できることに驚いた。「一部介助」という評価で漠然と捉えていたが、どの項目に問題があるかを特定することで治療の方向性が明確になると実感した
- 点数で変化を示すことの重要性を理解した。「中介助→軽介助」という言葉より数値の変化の方が伝わりやすく、客観的な指標が説明責任を果たす上でも必要だと感じた
- ブリッジ動作は「大臀筋強化」と「骨盤回旋改善」で同じ動作でも目的が異なることが印象的だった。PTとして何を目的に介入しているかを常に意識しないと、ただの筋トレになってしまうと反省した
- キッキングで足裏ではなく膝裏への抵抗が重要という点は、見た目だけでは判断できない。実習で実施する際には抵抗のかけ方を意識したい
- 1B抑制を使ったストレッチは理論と手技が結びついており理解しやすかった。ただし骨盤代償を抑えながら実施するには練習が必要だと感じた
- 「基礎なしで応用に走ると事故につながる」という言葉が印象に残った。かっこいい手技に目を奪われず、まずは安全なリスク管理ができる臨床家になることを優先したい
